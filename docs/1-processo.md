# Processo di sviluppo
Il processo adottato in questo progetto è tipo SCRUM.

Scrum è un processo di sviluppo di tipo Agile, iterativo ed incrementale che mira al miglioramento continuo tramite la consegna
di artefatti intermedi ma funzionanti.
